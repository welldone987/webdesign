# 仓库开发入口

本仓库是静态个人网站 monorepo。开始任何结构、实现、样式、测试、部署或依赖相关工作前，必须先阅读 `规范.md`。

`AGENTS.md` 只保留项目级入口要求；具体、可执行、需要统一遵守的规则维护在 `规范.md` 中。若开发过程中发现需要新增或调整统一规范，先更新 `规范.md`，再按新规范执行，避免项目内容不一致。

`README.md` 面向项目所有者，用于 review 项目结构、依赖关系、构建流程和当前状态。修改技术结构、依赖或构建流程时，应同步更新 `README.md`。

对于大任务或模糊要求，不要直接一次性执行。应先做必要澄清，并将任务拆解为多个可验证的小任务，分步实施、分步检查。
