Production rule

首屏建立一个固定设计舞台：桌面按 `1440 x 900`，移动按 `390 x 844` 的坐标比例生成同源布局，不在每个元素上临时猜百分比。

桌面 hover 预览的位置由目录行 DOM 的 `getBoundingClientRect()` 计算：预览中心 = hover 行中心；横向位置 = 目录列右侧固定间距。

移动端没有 hover 预览，但标题、目录和图库使用同一套舞台比例，保证体验一致。
